# 童锁交互模块交付包

这个目录是从当前原型中整理出的独立童锁模块，可直接交给他人部署到软件前端容器、WebView 或静态资源目录中。

## 目录说明

- `index.html`：童锁模块入口页
- `styles.css`：模块样式
- `script.js`：模块交互逻辑
- `README.md`：交付说明
- `DEPLOYMENT.md`：部署与接入说明
- `INTERACTION-SPEC.md`：交互流程说明
- `manifest.json`：交付元信息

## 当前已包含的童锁能力

- 童锁主页
- 正在熏听的时间、日期、资源信息展示
- 上滑触发家长解锁引导
- 9 点手势解锁面板
- 长按 2 秒进入紧急服务
- 长按 2 秒进入设置服务
- 宝宝信息卡页

## 本地打开方式

直接双击 `index.html` 即可预览。

如果对方的软件使用静态资源部署，也可以直接把本目录完整拷贝到目标项目中，通过相对路径访问 `index.html`。

## 默认交互规则

- 上滑距离大于 `70px` 才触发解锁引导
- 长按按钮 `2000ms` 进入服务页面
- 默认家长手势为 `1 -> 2 -> 3 -> 6`

## 可修改配置

如需改默认熏听内容或手势规则，可编辑 `script.js` 中以下内容：

- `trackInfo`
- `unlockPattern`

## 建议交付方式

优先交付压缩包 `../child-lock-package.zip`。

对方如果只做静态接入，解压后直接部署即可。
