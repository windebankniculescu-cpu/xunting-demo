# 童锁交互说明

## 核心流程

### 1. 进入童锁

- 默认进入页面即为童锁开启状态
- 中间区域提示“家长上滑后完成手势解锁”

### 2. 家长解锁

- 家长在底部中间区域上滑
- 弹出“先完成家长手势解锁”的提示
- 点击继续后进入 9 点手势面板
- 输入默认手势 `1 -> 2 -> 3 -> 6`
- 成功后提示“解锁成功，已返回播放主流程”

### 3. 紧急服务

- 点击左侧红色按钮时，仅提示“请长按”
- 长按 2 秒进入紧急服务页
- 紧急服务页支持进入宝宝信息卡

### 4. 设置服务

- 点击右侧按钮时，仅提示“请长按”
- 长按 2 秒进入设置服务页

## 页面清单

- 童锁主页
- 解锁提示弹窗
- 手势解锁页
- 紧急服务页
- 宝宝信息卡页
- 设置服务页

## 默认参数

- 解锁手势：`1 -> 2 -> 3 -> 6`
- 长按进入时间：`2000ms`
- 上滑阈值：`70px`
